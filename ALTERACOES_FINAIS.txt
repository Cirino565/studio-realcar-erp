OBSERVAÇÃO SOBRE O BUILD FINAL

O ambiente de revisão não conseguiu baixar os binários oficiais do Prisma por bloqueio de rede. Por isso, o build definitivo com o Prisma real deve ser executado no computador do projeto ou no Railway:

npm run build

Somente publique quando esse comando terminar com sucesso.

PUBLICAÇÃO

Opção manual:

npm run build
git add .
git commit -m "finalize premium crm experience"
git push

Opção assistida no Windows:

PowerShell -ExecutionPolicy Bypass -File .\FINALIZAR_E_PUBLICAR.ps1

No Railway, confira se o deploy utiliza o hash e a mensagem do último commit enviado. “Redeploy” de um deploy antigo repete o código antigo; para buscar o commit atual, use o deploy automático da branch conectada ou “Deploy Latest Commit”.

SEGURANÇA

O ZIP final não inclui .env ou .env.local. As credenciais existentes devem permanecer somente no computador do projeto e nas variáveis privadas do Railway.
